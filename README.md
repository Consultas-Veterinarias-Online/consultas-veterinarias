# Consultas Veterinarias Online

Sitios web y plataforma oficial para consultas veterinarias 24/7, triage de urgencias y orientación nutricional.

## 🚀 Características
- **Atención 24/7:** Orientación profesional sin esperas.
- **Diseño Responsivo:** Adaptado a dispositivos móviles y ordenadores.
- **Formulario Integrado:** Captura directa de datos de la mascota y motivo de consulta.

## 🛠️ Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
